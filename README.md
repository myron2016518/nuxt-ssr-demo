## 项目介绍

**项目中整合 vue + nuxt + axios + vuex + vue-router (nuxt 自带 vuex 和 vue-router)，一个基于 Nuxt 的服务器端渲染 Demo**

## 项目构建步骤

``` bash
# 安装依赖
$ npm install # Or yarn install

# 启动服务，访问 localhost:3001
$ npm run dev

# 生成环境打包
$ npm run build
$ npm start

# 打包需要部署的静态页面
$ npm run generate
```
